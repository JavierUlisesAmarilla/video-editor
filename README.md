## Setup

- composer update
- composer install
- Run MySQL server
- Create `laravel` database if not exists
- php artisan migrate
- php artisan db:seed
- php artisan storage:link
- npm install --legacy-peer-deps

## Run

- Open new command
- npm run dev
- Open new command
- php artisan serve
- Then visit localhost:8000
